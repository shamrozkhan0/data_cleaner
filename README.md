# Numpy Project

##### This program is made using `numpy` to clean messy data inside `messy_data.csv`, containing missing values, outliers, and inconsistencies.

---

### This program featured:

- Load `meesy_data.csv` as `numpy` `ndarray` 
- mark empty column with `numpy` `nan`
- fills the `nan` column with `mean` of the column using 'numpy.mean'
- Handles data outlier usinf `numpy.clip`
- Generates `cleaned_data.csv` file after successfully cleaning the data

---

### What best practices included:

- NumPy arrays (ndarray)
- Missing value handling with NaN
- Statistical operations(mean)
- Outlier handling
- File I/O with CSV
- Object-Oriented Programming (OOP)

### How To Run:

#### **Step 1:** `Clone This repository`
#### **Step 2:** `pip install numpy`
#### **Step 2:** `python cleaner.py`
